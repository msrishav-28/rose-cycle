RoseCycle is a Progressive Web Application (PWA) designed for comprehensive menstrual health tracking and wellness management. It offers users personalized insights, calendar tracking, daily logging, and educational resources through a modern, responsive interface.

## Quick Start

The frontend application is contained within the `frontend` directory.

```bash
# Navigate to the frontend application
cd frontend

# Install dependencies
npm install

# Start the development server
npm run dev

# Build for production
npm run build
```

## Features

- **Agentic Health Assistant**: Integrated chat interface for personalized health insights.
- **Cycle Calendar**: Advanced tracking for cycle phases, symptoms, and predictions.
- **Daily Logging**: Comprehensive daily logging for symptoms, mood, and physiological changes.
- **Wellness Education Hub**: Curated resources, guided breathwork, and structured wellness plans.
- **Reporting**: Ability to export detailed doctor reports.
- **Progressive Web App**: Offline capabilities and installable on modern devices.

## Tech Stack Overview

- **Core**: React 19
- **Routing**: React Router v7
- **Styling**: Tailwind CSS v4
- **Icons**: Lucide React
- **Build Tool**: Vite v8
- **PWA**: vite-plugin-pwa
